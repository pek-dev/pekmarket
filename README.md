# PekDev Market 🇧🇮

Marketplace moderne et professionnelle pour le Burundi, développée en PHP/MySQL avec Tailwind CSS.

## 🚀 Installation

### 1. Prérequis
- PHP 7.4 ou supérieur
- MySQL 5.7+ ou MariaDB 10.3+
- Apache/Nginx avec mod_rewrite
- Extension PHP : PDO, PDO_MySQL, GD

### 2. Installation

```bash
# Cloner ou copier le projet dans votre serveur web
cd /var/www/html/
git clone [votre-repo] pekdev-market
cd pekdev-market
